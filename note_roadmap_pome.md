# noteロードマップまとめ｜X集客からnote販売まで

出典：ぽめ「【完全保存版】副業初心者のためのClaude Code攻略大全」

---

## 全体像

```
X（気にならせる）→ note（納得させる）→ 商品（行動させる）
                    Claude Code（裏側サポート）
```

---

## 各媒体の役割分担

| 媒体 | 役割 | やること | やらないこと |
|-----|------|---------|------------|
| X | 人を集める | 興味・共感・続きが気になる状態を作る | 売り込む・長々と説明する |
| note | 価値を伝えて売る | 手法整理・考え方・信頼構築・購買意欲 | 情報の羅列・ただの日記 |
| Claude Code | 裏側サポート | 考える・調べる・整理する・改善する | これ単体でお金を生む |

---

## 基本の導線（最重要）

```
Xで気にならせる
  ↓
プロフィール・固定ポストからnoteへ誘導
  ↓
noteで「なるほど、だから必要なのか！」と納得させる
  ↓
商品を購入してもらう
```

**初心者がやりがちな失敗**
- Xでいきなり売り込む → 信頼がないと売れない
- X→noteのつながりが切れる → 導線が死ぬ
- noteが情報の羅列になる → 「欲しい」に変わらない

---

## 5つのAI役割分担

| 役割 | 仕事 | 使いどころ |
|-----|------|---------|
| **Strategist** | 方向性・誰向けか・何を売るか決める | 最初に必ずここから |
| **Researcher** | 読者の悩み・反応が取れる切り口・競合分析 | 発信・商品作り前 |
| **Writer** | 投稿文・note構成・商品説明文を作る | 方向性と材料がそろってから使う（いきなり丸投げしない） |
| **Analyst** | 反応を見て改善・勝ちパターンを見つける | 投稿・note公開後 |
| **Operator** | 作業整理・タスク分け・次のアクション明確化 | 迷ったとき・詰まったとき |

---

## 勝ち筋 vs 負け筋

| 勝ち筋 | 負け筋 |
|-------|-------|
| ターゲットがはっきりしている | 誰向けか曖昧 |
| 発信テーマがブレない | 毎日テーマが変わる |
| noteへの導線がある | Xだけ頑張って満足 |
| 商品がわかりやすい | 商品がない |
| 改善しながら続ける | AIに丸投げして終わる |

---

## 初月5万円の考え方

- 高額商品より「読みやすく・わかりやすく・すぐ使える」小さい商品1つから
- 1万円×5本でも、5千円×10本でもOK
- **大事なのは「1回でも売れる流れ」を作ること**
- 自動化・複数商品・複数SNSは最初はいらない

---

## 情報の蓄積（地味だが重要）

**フォルダ分け例**
- 投稿ネタ
- note原稿
- 商品案
- 読者の悩みメモ
- 反応が良かった投稿
- 改善メモ

一度うまくいった投稿・売れた言い回し・反応が良かった悩みを残しておくことで、毎回ゼロから考えなくて済む。

---

## CLAUDE.mdに入れておくと良いこと

- 自分が何を発信しているか
- 誰向けか
- 書く雰囲気・トーン
- 売りたい商品
- NGな表現

→ 毎回同じ説明をしなくて済む
